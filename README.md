# Car-RentalX 🚗

A modern and responsive car rental website built with HTML, CSS, and JavaScript.

## 🌟 Features

- Responsive navigation menu
- Interactive UI elements
- Smooth scrolling
- Mobile-friendly design
- Multiple sections:
  - Home
  - Rent
  - Services
  - Ride
  - Contact

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- RemixIcon
- ScrollReveal.js
